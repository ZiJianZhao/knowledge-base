# Knowledge Base

Personal knowledge base for algorithm engineer, maintained by Claude.

## Structure

```
knowledge-base/
├── papers/          # Paper notes (generated by paper-digest)
│   ├── llm-distillation/
│   ├── agent/
│   ├── reinforcement-learning/
│   └── _unsorted/
└── weekly/          # Weekly paper scan records (generated by weekly-papers)
```

## How it works

- Paper notes are auto-generated and committed by `paper-digest` skill
- Weekly scans are auto-generated and committed by `weekly-papers` skill
- Every write operation triggers an immediate commit + push
