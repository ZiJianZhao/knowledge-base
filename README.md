# Knowledge Base

Personal knowledge base for algorithm engineer, maintained by Claude.

## Structure

```
knowledge-base/
├── papers/          # Paper notes (generated by paper-digest)
│   ├── llm-distillation/
│   ├── agent/
│   ├── reinforcement-learning/
│   └── _unsorted/
├── weekly/          # Weekly paper scan records (generated by weekly-papers)
└── notes/           # Other research notes (hv-analysis reports, ad-hoc insights, etc.)
```

## How it works

- `papers/` — auto-generated by `paper-digest` skill, committed on every write
- `weekly/` — auto-generated by `weekly-papers` skill, committed on every write
- `notes/` — manually triggered: tell Claude "save this to knowledge-base", Claude writes + commits + pushes

## Saving to notes/

Tell Claude: "把这个存到 knowledge-base" or "save this to knowledge-base"

Claude will:
1. Determine the right filename (`YYYY-MM-DD-{topic}.md`)
2. Write the content to `notes/`
3. `git add → commit → push`
